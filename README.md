# Telegram Member Mass Adder Panel (SMM)

A modern, high-performance SMM panel specifically designed for Telegram member mass adding. Built with Next.js 14, Prisma, and Supabase.

## Features

- 🚀 **Fast Member Adding**: Automated Telegram member mass adding logic.
- 💰 **Wallet System**: Integrated balance management with simulated Stripe payments.
- 📊 **Order Tracking**: Real-time status updates for all your orders.
- 🔐 **Secure Auth**: Powered by NextAuth.js.
- 🎨 **Modern UI**: Clean dashboard built with Tailwind CSS.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Database**: PostgreSQL (via Supabase)
- **ORM**: Prisma
- **Auth**: NextAuth.js
- **Styling**: Tailwind CSS
